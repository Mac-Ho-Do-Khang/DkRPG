Detailed Description

Base class for all projectiles in the game, used by both player and enemy characters.

Note
Decide whether or not the projectile can pierce through multiple targets via ProjectileDamagePolicy:
  • EProjectileDamagePolicy::OnHit: explodes upon contacting 1st target
  • EProjectileDamagePolicy::OnBeginOverlap: persists through multiple targets
