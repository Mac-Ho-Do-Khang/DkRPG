Detailed Description

Rotates the AI-controlled character to face a specified target actor.

Note
The default BTTask RotateToFaceBBEntry requires the AI character to have
bUseControllerDesiredRotation = true;
bOrientRotationToMovement = false;
which is exactly the opposite of our settings for character movement, hence this custom task.
