Detailed Description

Base Gameplay Ability class providing common helper functionality.

Steps to configure a Gameplay ability
  • Create ability tag(s)
  • Subclass this class into a new Blueprint
  • Make corresponding Anim Montages(s)
  • Config Ability input actions
  • Grant the ability
