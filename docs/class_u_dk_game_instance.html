Detailed Description

The Game Instance class for this game.

Responsible for managing game-wide data and functionality, such as level references and music management.

Note
The project needs to be configured to use this class: Project Settings -> Game Instance -> Game Instance Class -> Assign the Blueprint subclass of this class.
