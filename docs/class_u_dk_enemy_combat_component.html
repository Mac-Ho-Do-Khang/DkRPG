EDkToggleDamageType ToggleDamageType )
overrideprotectedvirtual

Enables or disables (and clear OverlappedActors) collision for the enemy's body collision boxes (e.g., hands).

Currently, our Hero character does not use body collision boxes for melee attacks, only enemies do. So we implement this function here instead of in the base UDkPawnCombatComponent.
