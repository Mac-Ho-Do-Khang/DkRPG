Detailed Description

Base class for all weapons, providing mesh, collision, and damage detection functionality.

Note
All weapon-related task (e.g. spawning one, attaching to mesh, equip/unequip, etc.) are carried out via Gameplay abilities, each of which is a subclassed BP.

Callback when weapon collision box begins overlap with another actor/component.

Note
Overlaps are just volume intersections, not physical hits. SweepResult.bBlockingHit will be false, thus we will not be able to get an accurate hit location from the SweepResult. Thus, we will perform a line trace to get a more accurate hit location.
