Detailed Description

All gameplay tags used in the project.

Steps to configure inputs:
  • Native Gameplay Tags: Define input tags here.
  • Input config Data Asset: Map input tags to input actions.
  • Custom Input Component: Define input binding methods.
  • Binding Inputs: Define input callbacks
  • Assign Assets: Fill in assets needed in the editor.
Note
In order to play Gameplay Cues, add this config to DefaultGame.ini:
[/Script/GameplayAbilities.AbilitySystemGlobals]
GameplayCueNotifyPaths = "/Game/Blueprints/GameplayCues"
