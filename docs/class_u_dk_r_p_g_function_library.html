Computes the hit react direction tag based on the relative position of the Attacker to the Attackee.

Also outputs the angle difference in degrees between the Attackee's forward vector and the direction to the Attacker.

106{
107 checkf(Attacker && Attackee, TEXT("Attacker or Attackee passed to ComputeHitReactDirectionTag is null!"));
108
109 const FVector AttackeeForward{ Attackee->GetActorForwardVector() };
110 const FVector AttackeeToAttackerNormalized{ (Attacker->GetActorLocation() - Attackee->GetActorLocation()).GetSafeNormal() };
111
112 const double DotProduct{ FVector::DotProduct(AttackeeForward, AttackeeToAttackerNormalized) };
113 OutAngleDifference = UKismetMathLibrary::DegAcos(DotProduct); // always positive => cannot determine left/right
114
115 const FVector CrossProduct{ FVector::CrossProduct(AttackeeForward, AttackeeToAttackerNormalized) }; // order matters
116 if (CrossProduct.Z < 0)
117 {
118 OutAngleDifference = -OutAngleDifference;
119 }
120
121 // Determine direction tag based on angle difference
122 if (OutAngleDifference >= -45.f && OutAngleDifference <= 45.f)
123 {
124 return DkGameplayTags::Shared_Status_HitReact_Front;
125 }
126 else if (OutAngleDifference > 45.f && OutAngleDifference < 135.f)
127 {
128 return DkGameplayTags::Shared_Status_HitReact_Right;
129 }
130 else if (OutAngleDifference < -45.f && OutAngleDifference > -135.f)
131 {
132 return DkGameplayTags::Shared_Status_HitReact_Left;
133 }
134 else if (OutAngleDifference >= 135.f || OutAngleDifference <= -135.f)
135 {
136 return DkGameplayTags::Shared_Status_HitReact_Back;
137 }
138
139 return FGameplayTag{};
140}

Retrieves the display name of the key(s) mapped to the given IA within the specified IMC(s).

Parameters
InputMappingContextsThe list of IMCs to search for mappings to the given IA.
InputActionThe IA for which to retrieve the display key name(s).
Returns
Display names for the key(s). Chorded key are placed before the main key, separated by a " + " sign. If no mappings are found, returns "N/A".
Note
If exist multiple mappings for the same main/chorded IA, returns the first one found for each.
169{

173 {
174 for (const FEnhancedActionKeyMapping& Mapping : IMC->GetMappings())
175 {
176 if (Mapping.Action == InputAction) // Main IA found
177 {
178 // Retrieve key name from chorded action(s) (if any)
179 if (Mapping.Triggers.Num() > 0)
