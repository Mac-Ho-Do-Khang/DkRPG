Detailed Description

Carries additional payload for when the player interacts with a pick-up item.

This class is used as an optional payload carrier for the FGameplayEventData corresponding to the Player_Ability_PickUp_Stones tag, used to trigger related GAs.

FVector UDkPickUpItemInteractedAdditionalInfo::PickUpItemLocation
private

Location of the pick-up item interacted with.

This information is passed to related GA(s), which then is broadcasted to related UI component(s). The idea is to show the input key prompt at the location of the pick-up item after it is projected onto 2D screen space.
