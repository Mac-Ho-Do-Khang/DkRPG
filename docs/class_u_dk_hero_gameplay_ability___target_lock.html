Detailed Description

Enables the Hero to lock onto a target.

As the logic of target locking can be quite complex, this ability is implemented in C++ instead of Blueprints.

Note
Decrease the sensitivity of mouse input by decreasing the IMC lower threshold: IMC_TargetLock -> IA_SwitchTarget -> Add Modifier -> Dead Zone -> Set Lower Threshold up to 0.95.
Todo
[RECONSIDER] Using Mouse Button Up/Down or Mouse Wheel input instead of the same Mouse Axis 2D?
